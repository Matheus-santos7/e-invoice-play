# Frontend (`@e-invoice-play/frontend`)

Next.js 15 (App Router), Tailwind v4, simulador fiscal (UI migrada do antigo TanStack/Vite).

Na **raiz do monorepo**:

```bash
pnpm dev          # sobe este app (porta 3000)
pnpm dev:frontend # idem
pnpm build        # build frontend + backend
```

Rotas em `src/app/(app)/`. Componentes e mocks em `src/components` e `src/lib`.
